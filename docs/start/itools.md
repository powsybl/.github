# With itools

`itools` provides a command line interface to run Powsybl commands through a Windows or Linux terminal. For more information please refer to this [itools documentation](https://powsybl.readthedocs.io/projects/powsybl-core/en/stable/user/itools/index.html) to install, configure and use Powsybl via itools.

## Tutorials

For tutorial purpose you can clone the [tutorials](https://github.com/powsybl/powsybl-tutorials) repository and follow our itools tutorials ([itools tutorials documentation](https://powsybl.readthedocs.io/projects/powsybl-tutorials/en/latest/itools)).
